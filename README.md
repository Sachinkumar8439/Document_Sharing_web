## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Sachinkumar8439/Document_Sharing_web.git
   ```
2. go to project directory
    ```bash
    cd Document_Sharing_web
    ```
3. install packages
    ```bash
    npm install
    ```
4. start the development server
    ```bash
    npm start
    ```
